Class CriteriaFilter

This class is used to build CriteriaFilters, for use with Ethos resource using version 8 or later. Follows the builder pattern for easy construction of criteria filters. Inner class used for Building an instance of a CriteriaFilter, following the builder pattern. Use this Builder class to build CriteriaFilters. A CriteriaFilter can produce the following constructs, as examples:

Example of a CriteriaFilter with a SimpleCriteria: 

                      CriteriaFilter
       ______________________/\______________________
      /                                              \
     /                          CriteriaSet           \
    /            ___________________/\_________________\
   /            /                                      \\
  /            /                 SimpleCriteria         \\
 /            /      __________________/\________________\\
/            /      /                                    \\\
?criteria={"names":[{"firstName":"John","lastName":"Smith"}]}

Example of a CriteriaFilter with a NamedSimpleCriteria: 

                                           CriteriaFilter
           ______________________________________/\__________________________________
          /                                                                          \
         /                                           CriteriaSet                      \
        /           _____________________________________/\____________________________\
       /           /                                                                   \\
      /           /                                           NamedSimpleCriteria       \\
     /           /                         _________________________/\___________________\\
    /           /                        /                                               \\\
   /           /                        /Name                  SimpleCriteria             \\\
  /           /                        / / \       _____________________/\_________________\\\
 /           /                        / /   \    /                                         \\\\
?criteria={"alternativeCredentials": [{"type": {"id": "11111111-1111-1111-1111-111111111111"}}]}

Example of a CriteriaFilter with SimpleCriteria in multiple CriteriaSets: 

                                       CriteriaFilter
       _______________________________________/\__________________________________
      /                                                                           \
     /                        CriteriaSet                             CriteriaSet  \
    /          ___________________/\____________________           _________/\______\
   /          /                                         \         /                  \
  /          /                   SimpleCriteria          \       /     SimpleCriteria\\
 /          /          ________________/\_________________\     /        ______/\_____\\
/          /          /                                   \\   /       /              \\\
?criteria={"names":[{"firstName":"John","lastName":"Smith"}],"roles":[{"role":"student"}]}
