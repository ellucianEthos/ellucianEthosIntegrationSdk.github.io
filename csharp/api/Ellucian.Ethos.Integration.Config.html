Namespace Ellucian.Ethos.Integration.Config

Classes

EthosConfigurationClient

A client that can be used to call configuration endpoints in the Ethos Integration API.

SemVer

The SemVer class holds the semantic version for an Ethos resource. Implements the IComparable interface to enable easy sorting of this class by major/minor/patch values.

NOTE: Not all versions of Ethos resources are Semantic versions. Therefore, this class should only be used in support of those versions of resources following Semantic version notation.

SemVer.Builder

An inner static Builder class used for building the SemVer object with various criteria. This uses the builder fluent API pattern. All of the attributes in this Builder class correspond to the attributes in the containing SemVer class.
