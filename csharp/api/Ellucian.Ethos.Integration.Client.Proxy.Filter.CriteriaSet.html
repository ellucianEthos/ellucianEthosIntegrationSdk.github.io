Class CriteriaSet

This class is intended to be used internally by the SDK.

NOTE: This class is not intended to be used directly, but is used by the CriteriaFilter.Builder when constructing a CriteriaFilter. CriteriaFilters should be built using the CriteriaFilter.Build method.

A set of criteria containing a list of SimpleCriteria used for building a CriteriaFilter. The criteriaName is associated with the list of SimpleCriteria.
